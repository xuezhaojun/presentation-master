# Style: Default

Clean, balanced, and professional — suitable for most presentations.

## Theme
reveal.js theme: black

## Color Guidance
Choose a cohesive, professional palette that fits the topic. No fixed colors — pick what works best for the content. If the user specifies colors, use those.

## Typography
Use clean, readable fonts (system-ui or similar sans-serif).

## Layout
- Max bullets per slide: 5
- Prefer: text + code examples
- Transitions: slide

## Guidelines
- Use clear, professional tone
- Balance text and visuals
- Include speaker notes for complex slides
- Use consistent heading hierarchy
- Keep code examples concise and well-commented
