# Style: Vibrant

Colorful, icon-rich, and energetic — great for engaging audiences.

## Theme
reveal.js theme: moon

## Color Guidance
Use bold, saturated colors with strong visual impact. No fixed colors — pick a vibrant palette that matches the topic's energy. If the user specifies colors, use those.

## Typography
Use clean, readable fonts (system-ui or similar sans-serif). Bold headings encouraged.

## Layout
- Max bullets per slide: 5
- Prefer: icons, diagrams, and visual metaphors
- Transitions: convex

## Guidelines
- Use bold, energetic language
- Include emoji and icons where appropriate
- Use gradient backgrounds and accent colors liberally
- Add visual variety — mix slide layouts (text, image, split, full-bleed)
- Include speaker notes with talking points and audience engagement cues
- Use animations and fragment reveals to build up complex ideas
