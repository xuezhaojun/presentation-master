# Style: Minimalist

Sparse, whitespace-heavy, and focused — let the content breathe.

## Theme
reveal.js theme: white

## Color Guidance
Keep colors muted and restrained — high contrast, minimal palette. No fixed colors — adapt to the content. If the user specifies colors, use those.

## Typography
Use clean, readable fonts (system-ui or similar sans-serif). Prefer large font sizes.

## Layout
- Max bullets per slide: 3
- Prefer: single idea per slide, large text
- Transitions: fade

## Guidelines
- One concept per slide — avoid clutter
- Use generous whitespace and large font sizes
- Prefer full-bleed images over bullet lists
- Minimal speaker notes — the slides should speak for themselves
- Avoid decorative elements; keep it stripped down
- Use short, punchy phrases instead of full sentences
